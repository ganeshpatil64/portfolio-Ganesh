## Installation

Clone the repository:

```bash
git clone https://github.com/ganeshpatil64/portfolio-Ganesh.git
cd portfolio-Ganesh
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the environment:

Windows:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open:

```text
http://localhost:5000
```
